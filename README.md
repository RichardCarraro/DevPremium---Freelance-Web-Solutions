# 🚀 DevPremium - Freelance Web Solutions

Este é o repositório do portfólio oficial da **DevPremium**, desenvolvido para apresentar soluções web de alta performance, design premium e funcionalidades full-stack integradas.

## ✨ Funcionalidades

- **Design Premium & Responsivo:** Interface moderna construída com Tailwind CSS e animações fluidas via Framer Motion.
- **Portfólio Dinâmico:** Seção de trabalhos recentes focada em projetos reais (CineHub e Walletyx).
- **Backend Integrado (Express):** Servidor Node.js configurado para processar requisições de forma segura.
- **Sistema de Contato Real:** Integração Full-stack com a API do **Resend** para envio de e-mails diretamente para o proprietário.
- **Performance de Elite:** Estruturado com Vite para carregamento instantâneo e Core Web Vitals otimizados.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 19, TypeScript, Tailwind CSS, Framer Motion, Lucide React.
- **Backend:** Node.js, Express, Resend SDK, Zod (validação).
- **Ferramentas:** Vite, TSX, Dotenv.

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/devpremium.git
   cd devpremium
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env` na raiz do projeto e adicione sua chave do Resend:
   ```env
   RESEND_API_KEY=sua_chave_aqui
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
